### 袩芯谢褍褔懈褌褜袚褉邪薪懈褑褘袩邪褉褌懈褑懈懈

```1c

```

袩芯谢褍褔邪械褌 懈薪褎芯褉屑邪褑懈褞 芯 谐褉邪薪懈褑邪褏 (watermarks) 锌邪褉褌懈褑懈懈 褌芯锌懈泻邪.

**袘褉芯泻械褉** - 小褌褉芯泻邪. 袗写褉械褋 斜褉芯泻械褉邪 (薪邪锌褉懈屑械褉, "localhost:9092").

**袠屑褟孝芯锌懈泻邪** - 小褌褉芯泻邪. 袠屑褟 褌芯锌懈泻邪.

**袩邪褉褌懈褑懈褟** - 效懈褋谢芯. 袧芯屑械褉 锌邪褉褌懈褑懈懈.

**袙泻谢褞褔邪褌褜袙褉械屑褟袩芯褋谢械写薪械谐芯小芯芯斜褖械薪懈褟** - 袘褍谢械胁芯 (薪械芯斜褟蟹邪褌械谢褜薪褘泄, 锌芯 褍屑芯谢褔邪薪懈褞 `袥芯卸褜`). 袝褋谢懈 `袠褋褌懈薪邪`, 屑械褌芯写 写芯锌芯谢薪懈褌械谢褜薪芯 锌褉芯褔懈褌邪械褌 褋芯芯斜褖械薪懈械 锌芯 offset = `high_watermark - 1` 懈 胁械褉薪褢褌 械谐芯 胁褉械屑械薪薪褍褞 屑械褌泻褍 胁 锌芯谢褟褏 `last_message_timestamp` 懈 `last_message_timestamp_iso`. 袛谢褟 锌褍褋褌褘褏 锌邪褉褌懈褑懈泄 (`high_watermark == low_watermark`) 锌芯谢褟 斜褍写褍褌 锌褍褋褌褘屑懈. 袙泻谢褞褔械薪懈械 褎谢邪谐邪 写芯斜邪胁谢褟械褌 写芯锌芯谢薪懈褌械谢褜薪褘泄 round-trip 泻 斜褉芯泻械褉褍 懈 褔褌械薪懈械 芯写薪芯谐芯 褋芯芯斜褖械薪懈褟.

袦械褌芯写 胁芯蟹胁褉邪褖邪械褌 褋褌褉芯泻褍 JSON 褋 懈薪褎芯褉屑邪褑懈械泄 芯 谐褉邪薪懈褑邪褏 锌邪褉褌懈褑懈懈.

小褌褉褍泻褌褍褉邪:
+ topic - 袠屑褟 褌芯锌懈泻邪
+ partition - 袧芯屑械褉 锌邪褉褌懈褑懈懈
+ low_watermark - 袦懈薪懈屑邪谢褜薪褘泄 写芯褋褌褍锌薪褘泄 offset (oldest offset)
+ high_watermark - 袦邪泻褋懈屑邪谢褜薪褘泄 offset (offset 褋谢械写褍褞褖械谐芯 褋芯芯斜褖械薪懈褟)
+ message_count - 袣芯谢懈褔械褋褌胁芯 褋芯芯斜褖械薪懈泄 胁 锌邪褉褌懈褑懈懈 (high - low)
+ last_message_timestamp - *(褌芯谢褜泻芯 锌褉懈 `袙泻谢褞褔邪褌褜袙褉械屑褟袩芯褋谢械写薪械谐芯小芯芯斜褖械薪懈褟 = 袠褋褌懈薪邪`)* Unix-胁褉械屑褟 锌芯褋谢械写薪械谐芯 褋芯芯斜褖械薪懈褟 胁 屑懈谢谢懈褋械泻褍薪写邪褏. 袩褍褋褌邪褟 褋褌褉芯泻邪, 械褋谢懈 锌邪褉褌懈褑懈褟 锌褍褋褌邪 懈谢懈 斜褉芯泻械褉 薪械 锌褉械写芯褋褌邪胁懈谢 timestamp
+ last_message_timestamp_iso - *(褌芯谢褜泻芯 锌褉懈 `袙泻谢褞褔邪褌褜袙褉械屑褟袩芯褋谢械写薪械谐芯小芯芯斜褖械薪懈褟 = 袠褋褌懈薪邪`)* 袙褉械屑褟 锌芯褋谢械写薪械谐芯 褋芯芯斜褖械薪懈褟 胁 褎芯褉屑邪褌械 ISO 8601 UTC (`2026-05-12T10:23:45.678Z`). 袩褍褋褌邪褟 褋褌褉芯泻邪, 械褋谢懈 锌邪褉褌懈褑懈褟 锌褍褋褌邪 懈谢懈 斜褉芯泻械褉 薪械 锌褉械写芯褋褌邪胁懈谢 timestamp

> **小锌褉邪胁泻邪 芯 Watermarks:**
> - **Low Watermark** - 褋邪屑褘泄 褉邪薪薪懈泄 写芯褋褌褍锌薪褘泄 offset 胁 锌邪褉褌懈褑懈懈. 小芯芯斜褖械薪懈褟 褋 屑械薪褜褕懈屑懈 offset'邪屑懈 褍卸械 褍写邪谢械薪褘 褋芯谐谢邪褋薪芯 锌芯谢懈褌懈泻械 retention.
> - **High Watermark** - offset 褋谢械写褍褞褖械谐芯 褋芯芯斜褖械薪懈褟, 泻芯褌芯褉芯械 斜褍写械褌 蟹邪锌懈褋邪薪芯. 协褌芯 薪械 offset 锌芯褋谢械写薪械谐芯 褋芯芯斜褖械薪懈褟, 邪 offset + 1.
> - **Message Count** - 泻芯谢懈褔械褋褌胁芯 褋芯芯斜褖械薪懈泄, 写芯褋褌褍锌薪褘褏 写谢褟 褔褌械薪懈褟 (high_watermark - low_watermark).
